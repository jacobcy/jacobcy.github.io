# iWangPo Blog - Project Knowledge

**Type**: Hexo Static Blog  
**URL**: https://iwangpo.com  
**Stack**: Hexo 8.27.0 + NexT Gemini + Node.js

## Structure

```
/home/iwangpo/app/hexo/
├── _config.yml          # Hexo main config
├── _config.next.yml     # NexT theme config
├── source/
│   ├── _posts/         # Blog posts (23 articles)
│   └── about/          # About page
├── public/             # Generated static site
└── package.json        # Dependencies
```

## Key Locations

| Task | Location | Notes |
|------|----------|-------|
| Add post | `source/_posts/` | Markdown with YAML front-matter |
| Edit config | `_config.yml` | Hexo settings |
| Edit theme | `_config.next.yml` | NexT theme settings |
| View output | `public/` | Generated by `hexo generate` |
| SEO files | `source/` | robots.txt, etc. |

## Content Conventions

**Front-matter template:**
```yaml
---
title: Article Title

tags:

  - tag2
categories:
  - category
---
```

## Commands

```bash
cd /home/iwangpo/app/hexo

npx hexo server      # Dev server (localhost:4000)
npx hexo generate    # Build to public/
npx hexo clean       # Clear cache
npx hexo deploy      # Deploy (if configured)
```
