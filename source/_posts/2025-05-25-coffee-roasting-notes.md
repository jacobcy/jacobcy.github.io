> 🔥 "烘焙是咖啡风味的决定性时刻。"

## 🌱 从手冲到烘焙的进阶

距离我开始手冲咖啡已经3个月了。

在这个过程中，我逐渐意识到：**一杯好咖啡，50%取决于烘焙**。

无论你用多贵的设备、多熟练的手法，如果豆子烘焙得不好，一切都是白搭。

于是，我决定深入了解烘焙。

## 🔥 烘焙的基本原理

### 烘焙是什么

简单说，就是**加热生豆，引发美拉德反应和焦糖化反应**，发展出咖啡的风味。

### 烘焙过程中的关键阶段

```
温度
  │
230°C ├───────────────── 二爆开始
      │
210°C ├───────── 一爆结束/二爆开始
      │    ╱╲
200°C ├────╱  ╲──── 一爆开始（关键节点）
      │   ╱    ╲
180°C ├──╱      ╲
      │╱          ╲
150°C ├───────────── 转黄点
      │
 20°C ├───────────── 入豆
      └──────────────────────────→ 时间
      0    5    10   15   20  分钟
```

**关键节点**：

| 阶段 | 温度 | 现象 | 风味发展 |
|------|------|------|----------|
| **脱水期** | 20-150°C | 水分蒸发 | 青草味消失 |
| **转黄** | 150°C | 颜色变黄 | 面包香 |
| **梅纳反应** | 150-180°C | 颜色变褐 | 香气发展 |
| **一爆** | 195-205°C | 爆裂声 | 酸质、甜感 |
| **发展期** | 一爆后 | 风味定型 | 风味平衡 |
| **二爆** | 220-230°C | 第二次爆裂 | 苦味增加 |

## 🎯 烘焙度与风味

### 常见烘焙度

| 烘焙度 | 颜色 | 风味特点 | 适合的豆子 |
|--------|------|----------|-----------|
| **浅烘** | 肉桂色 | 酸质明亮，花香果香 | 非洲豆（耶加、肯亚） |
| **中浅烘** | 栗子色 | 酸甜平衡，层次丰富 | 中美豆（巴拿马、哥斯达黎加） |
| **中烘** | 深褐色 | 甜感突出，醇厚 | 南美豆（巴西、哥伦比亚） |
| **中深烘** | 深褐色 | 苦味明显，巧克力 | 曼特宁、巴西 |
| **深烘** | 近黑色 | 焦苦味，烟熏 | 意式拼配 |

### 我的偏好

经过大量杯测（品尝），我发现：

- **手冲**：喜欢中浅烘，保留产区特色
- **意式**：喜欢中深烘，醇厚适合加奶
- **冷萃**：喜欢中烘，甜感突出

## 🌍 产区特色与烘焙策略

### 非洲产区

**代表**：埃塞俄比亚、肯尼亚、卢旺达

**生豆特点**：
- 密度高
- 酸质明亮
- 花香、果香丰富

**烘焙策略**：
```
目标：浅烘到中浅烘（一爆后45-60秒出锅）

策略：
- 快烘（10-12分钟）保留花果香
- 发展期不要太长，避免酸质变钝
- 注意一爆密集度，非洲豆一爆通常很剧烈
```

**我的烘焙记录**：
```
豆子：埃塞俄比亚 耶加雪菲 G1 日晒
入豆温：180°C
发展时间：11分30秒
出锅温：205°C（一爆后50秒）
结果：花香明显，柑橘酸质，茶感余韵
评分：8.5/10
```

### 中南美洲产区

**代表**：哥伦比亚、巴拿马、哥斯达黎加、危地马拉

**生豆特点**：
- 平衡，酸甜适中
- 坚果、巧克力风味
- 醇厚度好

**烘焙策略**：
```
目标：中烘（一爆后60-90秒出锅）

策略：
- 中等烘焙时间（12-13分钟）
- 发展期适中，平衡酸甜
- 可以稍微发展焦糖化，增加甜感
```

**我的烘焙记录**：
```
豆子：哥伦比亚 蕙兰 水洗
入豆温：180°C
发展时间：12分45秒
出锅温：210°C（一爆后70秒）
结果：焦糖甜感，坚果香气，平衡度高
评分：8/10
```

### 亚洲产区

**代表**：印尼（曼特宁）、云南、越南

**生豆特点**：
- 醇厚度高
- 草本、香料风味
- 低酸

**烘焙策略**：
```
目标：中深烘（一爆后90-120秒或进入二爆）

策略：
- 可以慢烘（13-14分钟）
- 发展期较长，增加醇厚度
- 曼特宁可以进入二爆初段
```

**我的烘焙记录**：
```
豆子：印尼 曼特宁 G1 湿刨法
入豆温：170°C（密度较低，入豆温稍低）
发展时间：14分20秒
出锅温：218°C（一爆后110秒）
结果：草本香气，黑巧克力，醇厚饱满
评分：7.5/10（个人偏好不太喜欢曼特宁）
```

## 🎡 咖啡风味轮解读

### 风味轮的逻辑

```
                    花香
                     │
    果香 ────────────┼──────────── 甜味
    （柑橘/浆果）     │         （焦糖/蜂蜜）
                     │
    酸质 ────────────┼──────────── 坚果
                     │         （杏仁/榛果）
                     │
    草本植物 ─────────┴──────────── 巧克力/可可
                     │
                    香料
```

### 如何描述风味

**初学者误区**：
- ❌ "好喝" / "不好喝"
- ❌ "酸" / "不酸"

**进阶描述**：
- ✅ "明亮的柑橘酸质，类似橙子和柠檬"
- ✅ "有茉莉花香，余韵带有蜂蜜甜感"
- ✅ "醇厚度像牛奶巧克力，带有杏仁香气"

### 我的风味词汇库

**酸质类型**：
- 柑橘类：柠檬、橙子、柚子
- 浆果类：草莓、蓝莓、黑莓
- 核果类：桃子、杏子、李子

**甜感类型**：
- 焦糖类：焦糖、太妃糖、红糖
- 水果类：蜂蜜、枫糖、果酱
- 糖类：白糖、棉花糖

**香气类型**：
- 花香：茉莉花、玫瑰、薰衣草
- 果香：热带水果、莓果
- 其他：巧克力、坚果、香料

## 📊 烘焙曲线分析

### 理想的烘焙曲线

```
温度(°C)
  │
230 ├────────────────────────────────
    │                          ╱
220 ├────────────────────╱────╱
    │                ╱   ╱
210 ├──────────╱────╱───╱
    │      ╱   ╱   ╱
200 ├────╱────╱───╱
    │   ╱   ╱
190 ├──╱───╱
    │  ╱
180 ├─╱
    │╱
150 ├─────── 转黄点
    │
 20 ├────────────────────────────────
    └────────────────────────────────→ 时间(分钟)
    0   3   6   9  12  15  18
```

**关键指标**：

| 指标 | 理想范围 | 说明 |
|------|----------|------|
| **总烘焙时间** | 10-15分钟 | 太短风味发展不足，太长风味流失 |
| **转黄时间** | 4-6分钟 | 脱水期要充足 |
| **一爆时间** | 8-11分钟 | 决定酸质发展 |
| **发展时间比例** | 20-25% | 总时间的1/5-1/4 |

### 常见问题分析

**问题1：青草味/生味**
- 原因：脱水期不够，豆芯没熟
- 解决：延长脱水期，或提高入豆温

**问题2：酸味尖锐**
- 原因：发展期太短，或出锅太早
- 解决：延长发展期，或提高出锅温度

**问题3：苦味/焦味**
- 原因：烘焙过度，或升温太快
- 解决：降低出锅温度，或降低升温速率

**问题4：风味平淡**
- 原因：烘焙太均匀，没有突出特点
- 解决：根据豆子特点调整烘焙策略

## ☕ 杯测（Cupping）方法

### 杯测流程

```
1. 研磨：粗砂糖颗粒大小
2. 闻干香：记录干香
3. 注水：94°C热水，浸泡4分钟
4. 破渣：用勺子破开粉层，闻湿香
5. 撇渣：撇去表面浮渣
6. 品尝：用勺子舀取，用力吸入
7. 评分：根据风味、酸质、醇厚度等评分
```

## 🛒 生豆购买指南

### 购买渠道

| 渠道 | 优点 | 缺点 |
|------|------|------|
| **淘宝/天猫** | 方便，选择多 | 品质参差不齐 |
| **咖啡烘焙商** | 品质保证 | 价格较高 |
| **生豆商** | 新鲜，可选产地 | 起订量大 |
| **咖啡展会** | 可以杯测 | 机会少 |

### 选择建议

**新手建议**：
- 先买小样包（100-200g），多试几种
- 选择知名产区的豆子（耶加、肯尼亚、哥伦比亚）
- 注意烘焙日期，选1个月内的

**进阶选择**：
- 关注产季，买当季的豆子
- 尝试特殊处理法（日晒、蜜处理、厌氧）
- 建立与烘焙商的关系，获取好豆子
