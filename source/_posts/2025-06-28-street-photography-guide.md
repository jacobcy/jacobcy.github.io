> 📸 "最好的相机是你随身携带的那台。"

## 🚶 为什么开始街头摄影

2025年春天，我养成了一个新习惯：**周末不带目的地在城市里走走**。

不是为了去某个地方，只是走路、观察、拍照。

这个习惯源于一个想法：**我们日常经过的街角，可能藏着无数故事**。

## 🎯 街头摄影的魅力

### 什么是街头摄影

简单说：**在公共空间里，记录真实的瞬间**。

不是摆拍，不是风景照，而是**人与环境的互动**。

### 为什么用手机

| 设备 | 优点 | 缺点 |
|------|------|------|
| **手机** | 随时携带、不引人注意、即拍即修 | 画质受限、变焦差 |
| **相机** | 画质好、操控性强 | 笨重、引人注目 |

**街头摄影的黄金法则**：
> 拍到比拍好更重要。错过瞬间，再好的设备也没用。

## 🎨 构图技巧

### 基础构图法则

#### 1. 三分法则

```
┌─────┬─────┬─────┐
│     │     │     │
├─────┼─────┼─────┤ ← 地平线放这里
│     │     │     │
├─────┼─────┼─────┤
│     │     │     │
└─────┴─────┴─────┘
```

**应用场景**：
- 人物放在交叉点
- 地平线放在上1/3或下1/3

#### 2. 引导线

利用街道、栏杆、墙壁引导视线到主体。

**常见引导线**：
- 道路
- 栏杆
- 建筑边缘
- 阴影线条

#### 3. 框架构图

```
┌─────────────────┐
│   ┌─────────┐   │
│   │  主体   │   │ ← 门框、窗框、桥洞
│   └─────────┘   │
└─────────────────┘
```

### 街头摄影特有技巧

#### 层次感

街头摄影的灵魂是**层次感**——前景、中景、背景的叠加。

```
[前景]  模糊的人影/栏杆/植物
          ↓
[中景]  主体（人物/动作）
          ↓
[背景]  建筑/街道/天空
```

**拍摄技巧**：
- 靠近前景物体，制造虚化
- 等待人物走到合适位置
- 利用玻璃、水面制造反射

#### 光影对比

街头摄影最好的时间是**黄金时刻**（日出后/日落前1小时）。

**光影类型**：

| 类型 | 特点 | 拍摄技巧 |
|------|------|----------|
| **侧光** | 立体感强 | 人物侧面受光 |
| **逆光** | 剪影效果 | 对准亮部测光 |
| **斑驳光** | 树影/建筑阴影 | 等待人物经过光斑 |
| **反射光** | 橱窗/水面 | 寻找有趣反射 |

## 📸 实战场景与技巧

### 场景1：老街巷弄

**特点**：
- 生活气息浓厚
- 光影变化丰富
- 人物状态自然

**拍摄技巧**：
```
1. 提前到达，观察光线方向
2. 找好位置，耐心等待
3. 用长焦（2x-3x）压缩空间
4. 注意背景简洁
```

**样片参数**（参考）：
```
时间：下午4点
地点：老城区巷弄
设置：2x变焦, f/1.8, ISO 100
后期：提高阴影，降低高光，增加暖色
```

### 场景2：市集菜场

**特点**：
- 色彩丰富
- 人物互动多
- 烟火气最浓

**拍摄技巧**：
```
1. 俯拍摊位（表现丰富感）
2. 平拍交易瞬间（表现互动）
3. 特写手部动作（表现细节）
4. 注意征得同意（特别是正面特写）
```

**构图要点**：
- 利用蔬菜/水果的颜色对比
- 捕捉称重、递钱的瞬间
- 背景虚化突出主体

### 场景3：地铁站/公交站

**特点**：
- 人流密集
- 光影对比强烈
- 故事性强

**拍摄技巧**：
```
1. 利用车门的框架
2. 捕捉上下车的瞬间
3. 站台上的人物剪影
4. 车窗反射的双层影像
```

**注意事项**：
- 不要阻碍交通
- 注意人身安全
- 有些场所禁止拍照（看标识）

### 场景4：雨天街头

**特点**：
- 反射丰富
- 情绪氛围浓厚
- 行人状态特别

**拍摄技巧**：
```
1. 低角度拍地面反射
2. 利用雨伞作为前景框架
3. 雨丝效果（需要灯光背景）
4. 水花四溅的瞬间（高速快门）
```

**保护设备**：
- 用透明塑料袋包着手机
- 或买防水手机壳
- 雨后擦拭镜头（会有水珠）

## 🎨 后期调色思路

### 我的调色流程

```
1. 裁剪构图（二次构图）
2. 调整曝光（高光/阴影/对比度）
3. 白平衡（色温/色调）
4. HSL调整（单独颜色控制）
5. 分离色调（高光/阴影加色）
6. 锐化/降噪
7. 颗粒（胶片感可选）
```

### 不同风格调色

#### 风格1：胶片复古

```
特点：暖色调、低对比、颗粒感

Lightroom参数：
- 色温：+10
- 对比度：-20
- 高光：-50
- 阴影：+30
- 饱和度：-10
- 颗粒：+25
- 暗角：-20
```

**适合场景**：老街、咖啡馆、夕阳

#### 风格2：黑白高对比

```
特点：去色、强对比、突出光影

Lightroom参数：
- 黑白转换
- 对比度：+40
- 黑色色阶：-30
- 白色色阶：+20
- 清晰度：+20
- 锐化：+50
```

**适合场景**：建筑、光影强烈、人物剪影

#### 风格3：青橙色调

```
特点：电影感、冷暖对比

Lightroom参数：
- 色温：-5（偏冷）
- 分离色调：
  - 高光：色相30（橙色）
  - 阴影：色相210（青色）
- HSL：降低蓝/绿饱和度
- 提高橙/黄饱和度
```

**适合场景**：城市夜景、霓虹灯、雨天

## ⚠️ 街头摄影礼仪

### DO（应该做）

✅ **尊重被摄者**
- 如果被注意到，微笑示意
- 如果对方明确拒绝，立即删除
- 不拍摄儿童特写（避免误会）

✅ **融入环境**
- 穿着低调，不引人注目
- 动作自然，不要鬼鬼祟祟
- 可以带耳机，假装听音乐

✅ **了解法规**
- 有些场所禁止拍照（看标识）
- 尊重商家和私人空间
- 不拍摄敏感场所

### DON'T（不应该做）

❌ **不要 intrusive**
- 不要把镜头怼到人脸上
- 不要追着人拍
- 不要在人伤心/尴尬时拍

❌ **不要 commercial**
- 不要把街头照片用于商业用途
- 如果商用，需要肖像权授权
- 参加比赛/展览也要注意

## 🎯 练习计划

### 每周练习任务

**Week 1：观察训练**
- 每天拍10张照片
- 不追求质量，只练观察
- 记录光线、色彩、人物

**Week 2：构图训练**
- 每天拍1张三分法构图
- 1张引导线构图
- 1张框架构图

**Week 3：光影训练**
- 黄金时刻出门拍摄
- 练习逆光剪影
- 练习斑驳光影

**Week 4：故事训练**
- 拍一组有故事的照片（3-5张）
- 尝试组图叙事
- 记录拍摄时的想法

### 长期项目想法

1. **365天街头日记**
   - 每天1张，记录城市变化

2. **同一地点不同时间**
   - 选一个街角，不同时段去拍

3. **城市色彩地图**
   - 按颜色分类整理照片

4. **100个陌生人**
   - 拍摄100个陌生人的背影/侧面

## 💭 写在最后

街头摄影教会我一件事：**慢下来，看世界**。

我们习惯了赶路、刷手机、想着下一件事。

但当你举起相机，你会开始注意：

这些微小的观察，让平凡的日子变得有趣。

> "摄影不是关于你看到了什么，而是关于你如何看。"
